# 🎮 Hangman Game (Python)

A simple yet interactive text-based Hangman game built using Python.
This project was developed to practice core programming concepts such as loops, conditionals, functions, and user input handling.

---

## 🚀 Features

* 🎯 Difficulty Levels (Easy, Medium, Hard)
* 💡 Hint System (one-time hint per game)
* 🎨 ASCII-based Hangman UI
* 🔁 Replay Option
* ✅ Input Validation (handles invalid and repeated inputs)

---

## ▶️ How to Run the Game

1. Clone the repository:

```
git clone https://github.com/your-username/hangman-python-game.git
```

2. Navigate to the project folder:

```
cd hangman-python-game
```

3. Run the game:

```
python hangman.py
```
